# JavaScript, TypeScript study

Folder to study JavaScript, TypeScript, and related technologies.

## Node version manager installation

```
$ nvm install node
$ nvm use node 16
```

## NodeJS installation

```
$ brew install node
$ node file.js
$ npx ts-node file.ts
```

## Deno installation

```
$ brew install deno
```

## Bun installation

```
$ brew tap oven-sh/bun
$ brew install bun
```
