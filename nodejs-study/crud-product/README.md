# Project
> Project to study Node

# Technologies

- Node
- Express (micro-framework)
- Nodemon (hot deploy)
- Mongoose (ORM)

# Create the project

```shell script
$ mkdir <project>
$ npm init
$ eslint --init
```

# Dependencies

```shell script
$ npm install express mongoose mongoose-paginate require-dir cors
$ npm install nodemon eslint -D
```

# Build and launch application

```shell script
$ npm install
$ npm run dev OR node server.js
```
